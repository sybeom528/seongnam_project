# 성남시 자영업자 문제 분석 계획 보고서

**생성일**: 2026-04-07
**프로젝트**: 2026 성남시 공공데이터 활용 시각화 경진대회
**마감일**: 2026-05-13 (수) 18:00
**접근 방식**: Bottom-up (데이터 탐색 → 문제 발견 → 문제 정의 → 해결 제안)

## 1. 컨텍스트 요약

### 1.1 프로젝트 배경
- 성남시 공공데이터 활용 시각화 경진대회 참가
- 주제: 성남시 자영업자들이 겪는 문제를 데이터로 파악하고 해결 방안 제시
- 성남시 3개 구(분당구, 수정구, 중원구) 간 비교 분석을 통해 지역별 격차와 특성 파악

### 1.2 사용 가능 데이터 (총 1,061개 CSV, 2023.01 ~ 2025.12)

| 대분류 | 데이터셋 | 핵심 내용 | 자영업 관련도 |
|--------|----------|-----------|-------------|
| **카드** | 가맹점 정보 (mer_s) | 개업·휴업·폐업 수, 영업기간별 가맹점 수 | ⭐⭐⭐ 핵심 |
| **카드** | 매출 (day) | 일별·시간대·성별·연령별 카드 매출액/건수 | ⭐⭐⭐ 핵심 |
| **기업** | 법인 기업 (cnt) | 업종별 법인 기업 현황 | ⭐⭐ 보조 |
| **기업** | 신규 기업 (new) | 업종별 신규 법인 수 | ⭐⭐ 보조 |
| **기업** | 기업 이전 통계 | 기업의 전입/전출 이동 | ⭐ 참고 |
| **신용** | 신용정보 (DM_STAT) | 대출·카드사용·신용점수·소득 | ⭐⭐ 보조 |
| **신용** | 전입/전출 (IN/OUT) | 인구 이동 + 소득·신용 정보 | ⭐⭐ 보조 |
| **신용** | 전이 (CHANGE/L) | 신용등급 변화 추이 | ⭐ 참고 |
| **신용** | 이동통계 (WORK_STAT) | 거주지-직장 이동 패턴 | ⭐ 참고 |
| **통신** | T4 (유입) | 목적별·성연령별 유동인구 유입 | ⭐⭐⭐ 핵심 |
| **통신** | T10 (유출) | 교통수단별 유동인구 유출 | ⭐⭐ 보조 |
| **통신** | T20 (탄소배출) | 이동거리·탄소배출 | ⭐ 참고 |
| **통신** | T5~T27 (기타) | 행정동별 세부 유동인구 데이터 | ⭐⭐ 보조 |

### 1.3 분석 범위
- **공간**: 성남시 전체 (분당구·수정구·중원구 비교)
- **시간**: 2023년 1월 ~ 2025년 12월 (약 3년간 추이)
- **대상**: 자영업자/소상공인 (카드 가맹점 기반)

## 2. Bottom-up 분석 워크플로

### Phase 1: 데이터 탐색 및 문제 발견 (EDA)
> **목표**: 데이터를 통해 자영업자가 겪는 문제를 스스로 발견한다

#### Step 1-1: 데이터 병합 및 전처리
- **사용 스킬**: `multi-file-merge` → `data-quality-audit` → `data-preprocessing`
- **작업 내용**:
  - 각 카테고리별 월별 CSV 파일을 하나의 통합 데이터셋으로 병합
  - 결측값, 이상치, 데이터 타입 확인 및 정제
  - 공통 키(시군구코드, 행정동코드, 기준년월) 기준으로 데이터 연계 준비
- **예상 소요**: 2~3시간

#### Step 1-2: 탐색적 데이터 분석 (EDA)
- **사용 스킬**: `programmatic-eda` → `visualization-builder`
- **핵심 탐색 질문**:
  1. 성남시 자영업 가맹점의 개업/폐업 추이는 어떤가? (생존율은?)
  2. 업종별·지역별 매출 트렌드에 차이가 있는가?
  3. 유동인구 변화와 매출 변화 사이에 관계가 있는가?
  4. 구별(분당·수정·중원) 자영업 환경에 어떤 격차가 있는가?
  5. 특정 업종이나 지역에서 급격한 변화가 관찰되는가?
- **예상 소요**: 3~4시간

#### Step 1-3: 문제 가설 수립
- **EDA 결과를 바탕으로 3~5개 핵심 문제 가설 도출**
- 예시 가설 (데이터 확인 전 초기 방향):
  - H1: "수정구·중원구 자영업 폐업률이 분당구 대비 높고, 영업 생존기간이 짧다"
  - H2: "특정 업종(예: 음식점)에서 매출 감소가 심화되고 있다"
  - H3: "유동인구 감소 지역과 매출 감소 지역이 일치한다"
  - H4: "신규 창업 대비 폐업이 더 빠르게 증가하고 있다"
- **예상 소요**: 1시간

---

### Phase 2: 문제 정의 및 심화 분석
> **목표**: 발견된 문제를 통계적으로 검증하고 원인을 파악한다

#### Step 2-1: 통계 검정
- **사용 스킬**: `stats-basic` → `stats-advanced`
- **작업 내용**:
  - 구별 매출 차이 검정 (독립표본 t-검정, ANOVA)
  - 유동인구와 매출의 상관분석
  - 폐업률의 시계열 추세 검정
- **예상 소요**: 2~3시간

#### Step 2-2: 시계열 분석
- **사용 스킬**: `time-series-analysis`
- **작업 내용**:
  - 매출 추세·계절성 분해
  - 업종별·지역별 매출 예측 모델
  - 이상 시점(급격한 변화) 탐지
- **예상 소요**: 2~3시간

#### Step 2-3: 세분화/군집 분석
- **사용 스킬**: `segmentation-analysis` 또는 `ml-unsupervised`
- **작업 내용**:
  - 행정동별 자영업 환경을 군집화 (매출·유동인구·폐업률 등 기준)
  - 위험 지역/업종 식별
  - 성공·실패 패턴 도출
- **예상 소요**: 2~3시간

---

### Phase 3: 해결 방안 도출 및 결과물 작성
> **목표**: 데이터 기반 해결 제안과 경진대회 제출물을 완성한다

#### Step 3-1: 인사이트 종합 및 해결 방안 제안
- **사용 스킬**: `insight-synthesis` → `data-narrative-builder`
- **작업 내용**:
  - 핵심 발견사항을 실행 가능한 제안으로 변환
  - 성남시 정책 제안 형태로 정리
  - 데이터 스토리라인 구성
- **예상 소요**: 2시간

#### Step 3-2: 최종 결과물 제작
- **결과물 목록**:
  1. **인터랙티브 대시보드 (HTML/React)** — 핵심 시각화 및 인사이트
  2. **분석 보고서 (MD → DOCX)** — 상세 분석 과정 및 결과
  3. **Jupyter Notebook** — 재현 가능한 분석 코드
  4. **발표자료 (PPTX)** — [붙임3] 양식 기반 공모전 제출용
- **사용 스킬**: `visualization-builder`, `docx`, `pptx`
- **예상 소요**: 4~5시간

#### Step 3-3: 품질 검증
- **사용 스킬**: `analysis-qa-checklist`
- **작업 내용**: 분석 완전성, 시각화 정확성, 제출 요건 충족 여부 확인
- **예상 소요**: 1시간

## 5. 의존 관계 및 리스크

### 의존 관계
- Phase 2는 Phase 1 완료 후 진행 (EDA 결과에 따라 분석 방향 조정)
- 결과물 제작은 Phase 2 완료 후 진행
- 발표자료는 모든 분석 완료 후 마지막에 제작

### 리스크 및 대응
| 리스크 | 대응 방안 |
|--------|----------|
| 데이터 간 조인 키 불일치 | 코드 매핑 테이블 수동 생성, 근사 매칭 |
| 특정 행정동의 데이터 부족 | 시군구(구) 수준으로 집계하여 분석 |
| 분석 시간 초과 | Phase 2에서 가장 유의미한 가설 1~2개에 집중 |

### 분석자 역량
- Python pandas/matplotlib/seaborn 중급 수준
- 고급 기법(scikit-learn, statsmodels 등) 학습에 적극적
- 실무 결과물 도출에 필요하다면 스킬 수준 제한 없이 활용 가능
- → ML 군집분석, 시계열 예측 등 고급 분석 기법도 적극 활용 예정
