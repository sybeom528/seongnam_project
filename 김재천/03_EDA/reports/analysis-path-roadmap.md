## 분석 개요

| 항목 | 내용 |
|------|------|
| 분석 목적 | 성남시 자영업자 문제를 데이터에서 발견하고 해결 방안 제시 |
| 데이터 현황 | 1,061개 CSV, 4개 카테고리(기업·신용·카드·통신), 2023~2025년 |
| 데이터 유형 | 월별 시계열 집계 데이터 |
| 추천 경로 | **경로 G(완전 탐색) → D(원인분석) + E(추이분석) + C(군집화) 복합** |
| 분석 단위 | 성남시 3개 구(분당구·수정구·중원구) 비교 |

## 핵심 발견: 데이터 프로파일링 결과

### 자영업 분석에 핵심적인 데이터 포인트

1. **가맹점 데이터에서 "폐업 > 개업" 신호 감지**
   - 2023년 2월 기준 1개월 데이터만으로도: 개업 1,036건 < 폐업 1,270건 < 휴업 44,619건
   - 휴업이 폐업의 35배 → 잠재적 폐업 위험군이 대량 존재

2. **분당구 가맹점 수가 수정구+중원구 합산과 비슷**
   - 분당구(41135): 29,706건 vs 중원구(41133): 15,382건 vs 수정구(41131): 14,907건
   - 구별 자영업 환경 격차 존재 가능성

3. **전입/전출 데이터에 자영업자 수(SELF_ENT) 컬럼 존재**
   - 자영업자의 지역 이동을 직접 추적 가능 → 젠트리피케이션 분석 연계 가능

4. **매출 데이터가 매우 세밀** (일별·시간대·성별·연령·업종)
   - 9개 업종 대분류 중 "음식", "소매/유통", "생활서비스"가 자영업 핵심 업종

## 단계별 실행 계획

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 Phase 1: 데이터 준비 및 탐색 (EDA)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1-1: multi-file-merge (데이터 병합)
  ├─ 대상: 카드 가맹점 36개 → 1개 통합 CSV
  ├─ 대상: 카드 매출 36개 → 구별·업종별 월간 집계 CSV
  ├─ 대상: 기업 법인+신규 72개 → 1개 통합 CSV
  ├─ 대상: 신용 전입+전출+신용정보 108개 → 구별 월간 집계
  ├─ 대상: 통신 T4(유입) 36개 → 구별·시간대 집계
  └─ 결과: 5~6개의 분석용 통합 데이터셋

Step 1-2: column-navigator (컬럼 구조 파악)
  ├─ 대상: 신용정보(39컬럼), 전입/전출(41컬럼)
  └─ 결과: 분석에 필요한 핵심 컬럼 목록 확정

Step 1-3: programmatic-eda (탐색적 분석)
  ├─ 핵심 분석:
  │   ├─ [가맹점] 구별·업종별 개업/폐업/휴업 추이
  │   ├─ [매출] 구별·업종별 월간 매출 추이
  │   ├─ [유동인구] 구별 유동인구 변화 추이
  │   ├─ [신용] 구별 소득·대출·신용점수 분포
  │   └─ [전입전출] 자영업자(SELF_ENT) 이동 패턴
  └─ 결과: 문제 가설 3~5개 도출

Step 1-4: data-quality-audit (품질 점검)
  └─ 결측값, 이상치, 코드 불일치 확인

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 Phase 2: 문제 정의 및 심화 분석
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 2-1: stats-basic → stats-advanced (통계 검정)
  ├─ 구별 매출 차이 검정 (ANOVA)
  ├─ 유동인구 ↔ 매출 상관분석
  ├─ 폐업률의 시계열 추세 검정
  └─ 자영업자 전입/전출 비율 비교

Step 2-2: time-series-analysis (시계열 분석)
  ├─ 매출 추세·계절성 분해
  ├─ 업종별·지역별 매출 예측
  └─ 이상 시점(급격한 변화) 탐지

Step 2-3: root-cause-investigation (원인 분석)
  ├─ 폐업률 증가의 원인 드릴다운
  ├─ 매출 감소 지역의 복합 원인 분석
  └─ 유동인구 변화 → 매출 변화 인과관계 탐색

Step 2-4: ml-unsupervised (군집 분석)
  ├─ 행정동별 자영업 환경 군집화
  │   (매출·유동인구·폐업률·신용 등 다차원 기준)
  ├─ 위험 지역/건강한 지역 클러스터 식별
  └─ segmentation-analysis로 군집 해석

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 Phase 3: 인사이트 도출 및 결과물 제작
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 3-1: insight-synthesis (인사이트 종합)
  └─ 핵심 발견사항을 실행 가능한 정책 제안으로 변환

Step 3-2: data-narrative-builder (스토리라인 구성)
  └─ 데이터 기반 내러티브로 분석 흐름 구성

Step 3-3: visualization-builder (시각화 제작)
  └─ 인터랙티브 HTML 대시보드 생성

Step 3-4: analysis-documentation (문서화)
  ├─ Jupyter Notebook (재현 가능한 분석 코드)
  ├─ 분석 보고서 (DOCX)
  └─ 발표자료 (PPTX, [붙임3] 양식 기반)

Step 3-5: analysis-qa-checklist (품질 검증)
  └─ 분석 완전성·정확성 최종 검증
```

## 추가 데이터 수집 계획

Phase 1 EDA 이후, 문제 가설에 따라 다음 외부 데이터를 추가 수집:

| 데이터 | 출처 | 용도 |
|--------|------|------|
| 상권 정보 | 소상공인시장진흥공단 | 상권 활성화 지수, 업종 밀집도 |
| 부동산 임대료 | 국토교통부 실거래가 | 젠트리피케이션 분석 |
| 인구 통계 | 성남시 통계 | 주민등록 인구, 연령 구조 |
| 배달앱 데이터 | (공개 데이터 있는 경우) | 온라인 매출 보완 |

## 의사결정 포인트

Phase 1 완료 후 아래 분기점에서 방향을 결정합니다:

```
EDA 결과
  │
  ├─ 폐업률이 핵심 문제 → root-cause-investigation 강화
  │                       + 부동산 임대료 데이터 추가
  │
  ├─ 매출 양극화가 핵심 → segmentation-analysis 강화
  │                       + 업종별 심층 비교
  │
  ├─ 유동인구 감소가 핵심 → time-series-analysis 강화
  │                         + 통신 데이터 세밀 분석
  │
  └─ 복합 문제 → 위 세 가지 모두 수행
               (다만 각각의 깊이는 조절)
```
